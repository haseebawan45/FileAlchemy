FileAlchemy - File Conversion Suite
=============================================

Thank you for using FileAlchemy!

Getting Started:
---------------
1. Run "FileAlchemy.bat" to start the application
2. Your web browser will open automatically to http://localhost:8001
3. Upload files and select your desired conversion format
4. Download your converted files

Supported Formats:
-----------------
* Documents: PDF, DOCX, XLSX, PPTX, TXT, HTML, Markdown
* Images: JPG, PNG, WEBP, GIF, BMP, TIFF, ICO, HEIC
* Video: MP4, AVI, MOV, MKV, WEBM, FLV, WMV
* Audio: MP3, WAV, AAC, OGG, FLAC, OPUS, AC3, M4A, ALAC
* Data: CSV, JSON, XML

Special Features:
---------------
* HEIC Image Support: Convert iPhone HEIC photos to standard formats
* Audio Extraction: Extract audio tracks from video files
* OCR: Extract text from images
* Image Compression: Reduce image file sizes

Troubleshooting:
--------------
* If the application doesn't start, ensure you have write permissions in the application folder
* For video and audio conversions, FFmpeg must be properly installed
* Check the console window for any error messages
